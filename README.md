# Introduction

This repository is for a senior project for Volume 3 of the ACME core. In this project we examine various sources of climate-related data in order to determine trends and make predictions about the future state of our climate. The primary research question we seek to answer is to determine how worried we should be about global climate change, and what the next few decades may look like.

# Summary

All of the Python files we used for data analysis are contained in the root of this repository. The `presentation` folder contains the LaTeX files for the presentation of our results, and the `tex_dir` folder contains the LaTeX files for our final writeup.
